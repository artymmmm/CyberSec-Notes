- [Kali Linux](https://www.kali.org/): дистрибутив Linux на основе Debian для пентеста, форензики, реверс-инжиниринга
- [Athena OS](https://athenaos.org/): дистрибутив Linux на основе Arch/Nix для пентеста
- [FlareVM](https://github.com/mandiant/flare-vm): дистрибутив Windows для ВМ, использующийся для статического и динамического анализа ВПО
