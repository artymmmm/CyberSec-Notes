## Места для поиска уязвимостей
- [База знаний Common Vulnerabilities and Exposures (CV)](https://cve.mitre.org/)
- [Платформа Vulners](https://vulners.com/)
- [База данных эксплойтов ExploitDB](https://www.exploit-db.com/)
- [Платформа управления уязвимостями и анализа угроз VulDB](https://vuldb.com/)
- Metasploit:  `search <название ПО>`
## Инструменты
- ### Сканеры уязвимостей
	- [vulscan](https://github.com/scipag/vulscan): `nmap -sV --script=vulscan/vulscan.nse <target> > vulscan_res.txt` (вывод в файл)
	- OpenVAS
- ### Эксплуатация уязвимостей
	- #### Metasploit
		1. `search <название ПО>` (поиск эксплойтов)
		2. `use <exploit>` (выбор эксплойта)
		3. `info <exploit>`, `options <exploit>` (информация и параметры эксплойта)
		4. `set <option> <value>` (установка значений параметров)
		5. `set payload <value>` (установка полезной нагрузки)
		6. `exploit` (выполнение эксплойта)
	- #### CrackMapExec
		- CME используется для атак на Windows
