## Формат файла
Формат файла можно определять по следующим признакам:
- разрешение файла
- сигнатура файла (магические числа) - последовательность 2-4 байтов в начале файла. Проверить можно следующим образом: `file <file>`; получить первые байты с помощью команды `xxd <file> | head`, произвести поиск по [GCK File Signature Table](https://filesig.search.org/)
## Метаданные
### Временные метки файлов
С помощью временных меток возможно обнаружение переименование файлов, копирование с того же диска, копирование с другого диска, использования инструмента "steghide". Описание маркеров действий с файлом: https://ctf101.org/forensics/what-is-metadata/
